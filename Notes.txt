Apa yang harus disiapkan sebelumnya:
1. PHP 8.0
2. Composer
3. PostgreSQL

Memulai pertama kali:
1. composer install
2. npm install
3. switch-db.bat local (apabila menggunakan postgresql local)
   ./switch-db.sh local (apabila command diatas tidak bekerja, biasanya di Mac/Linux)
   switch-db.bat supabase (apabila menggunakan supabase)
   ./switch-db.sh supabase (apabila command diatas tidak bekerja, biasanya di Mac/Linux)
4. docker-compose up -d --build

Memulai setelah yang pertama:
Windows - Switch to LOCAL       = docker-compose up -d
Windows - Switch to SUPABASE    = docker-compose up -d
                                  docker-compose up -d queue (apabila ingin manual)

Import besar via CLI:
  docker-compose exec app php artisan pendamping:import "database/CSV Data Pendamping.csv"

Cek status di docker: docker-compose ps
