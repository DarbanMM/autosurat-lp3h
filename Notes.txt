What have to be prepared:
1. PHP 8.0
2. Composer
3. PostgreSQL

start apps:
Windows - Switch to LOCAL       = switch-db.bat local
                                  docker-compose up -d
Windows - Switch to SUPABASE    = switch-db.bat supabase
                                  docker-compose up -d
                                  docker-compose up -d queue   (for background import)

Import besar via CLI:
  docker-compose exec app php artisan pendamping:import "database/CSV Data Pendamping.csv"
Mac/Linux - Switch to LOCAL     = ./switch-db.sh local
                                  docker-compose up -d
Mac/Linux - Switch to SUPABASE  = ./switch-db.sh supabase
                                  docker-compose up -d

after that, run "npm run dev" for UI

Checking status for docker: docker-compose ps

How to access for local: http://localhost
